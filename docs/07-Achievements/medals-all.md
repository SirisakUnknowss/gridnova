## สรุปไฟล์ที่ต้องแก้เมื่อ implement

| ไฟล์ | การเปลี่ยนแปลง |
|---|---|
| `supabase/migrations/YYYYMMDD_achievement_250.sql` | สร้างใหม่ — insert 250 rows + rewrite `check_and_grant_achievements` |
| `src/ui/views/achievements.ts` | เพิ่ม `badge_mission` field, เปลี่ยน render ให้แสดง 5 card/group |
| `supabase/seed.sql` | อัปเดต achievement seed ให้ตรงกับ 250 rows |

**เมื่อ implement ภารกิจไหนแล้ว เปลี่ยน status ในตารางนี้เป็น ✅**
